# Python Playground

Python Playground is a GUIn based minigames application built using **Python**, **Tkinter**, and **CustomTkinter**.  
The project contains multiple interactive games combined into a single desktop application.

This project was created as a learning exercise to practice:
- Python programming
- Object-Oriented Programming (OOP)
- GUI development using Tkinter

---

##  Games Included

- **Guess the Word**
- **Quiz Game**
- **Rock, Paper, Scissors**
- **Typing Speed Game**
- **Word Scramble**
- **Number Guessing Game**

Each game is implemented as a separate class and can be accessed from the main menu.

---

##  Technologies Used

- Python 3
- Tkinter
- CustomTkinter
- Random & Time modules

---

## ▶️ How to Run

1. Make sure Python is installed.
2. Install CustomTkinter:
   pip install customtkinter
